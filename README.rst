=====================
mingtak.backToBackend
=====================

Features
--------

- Conctrl page redirect to backend
- Back to portal root when logged in and logged out

Installation
------------

Install mingtak.backToBackend by adding it to your buildout::

    [buildout]

    ...

    eggs =
        mingtak.backToBackend


and then running ``bin/buildout``
